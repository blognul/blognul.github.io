使用Calibre-Web更好的在线管理你的图书

我是单独空出了一台老机器作为我的服务器使用，系统使用Ubuntu18，硬盘为4T东芝机械硬盘，后续还会作为简单的私有网盘使用

项目地址：https://github.com/janeczku/calibre-web

可以参考项目的WIKI来使用docker进行搭建，这里我是直接搭建了

1
2
3
4
5

git clone https://github.com/janeczku/calibre-web

cd calibre-web

pip3 install --target vendor -r requirements.txt

这里几点需要注意吧，首先Ubuntu18是没有默认安装pip3的，并且在使用pip3安装的时候需要加system参数

1
2
3

sudo apt-get install python3-pip

pip3 install --target vendor -r requirements.txt --system

此时，其实calibre-web服务已经可以启动，但是为了能够正常使用calibre-web还需要安装一些其他的程序

1
2
3
4
5

# 为了能够使calibre-web提取图书中的封面图片
sudo apt-get install imagemagick

# 为了初始化书库功能且支持转码功能
sudo apt-get install calibre

在启动calibre-web之前你需要先初始化一个书库

1
2
3
4
5

# 建立书库
calibre --with-library=/path/

# 添加书籍
calibre add book.name

可能在建立书库的时候会出现报错，可以暂且忽略，添加时没问题的，然后启动calibre-web

1
2

cd calibre-web
python3 cps.py

此时访问8083端口即可看见配置界面，要求填入一个书库地址，就填写刚刚建立的书库即可

应为界面可以调整成为中文，所以剩下的操作都比较简单，自行摸索即可

需要注意的时，默认没有开启上传功能，你需要在 权限管理->编辑基本配置 中进行勾选，另外为了支持格式转换功能，需要在 编辑基本配置->拓展程序配置 中配置Calibre电子书转换器路径(使用 whereis ebook-convert 即可)，一般都是/usr/bin/ebook-convert吧。。

# .service前的服务名字你可以自己定
sudo vim /etc/systemd/system/cps.service

# 内容
[Unit]
Description=Calibre-Web

[Service]
Type=simple
User=root
ExecStart=/usr/bin/python3 /calibre-web-path/cps.py
WorkingDirectory=/calibre-web-path

[Install]
WantedBy=multi-user.target

# 启动服务
sudo systemctl start cps
# 开机启动
sudo systemctl enable cps

另外，calibre-web同时提供了OPDS服务，我使用IOS的kybook3程序对接OPDS，看书还是比较爽的，另外Kindle设备理所当然也是支持OPDS的，具体请自行使用
