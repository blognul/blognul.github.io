Tmux是一款我非常喜欢的终端复用工具，本文章记录了简单的使用方法，便于自己记忆以及新手的入门

如果使想方便使用的话，建议直接使用插件，然后其余的再进行配置的定制，之后我会附上我的tmux配置文件

常用配置

感谢原作者的文章： http://louiszhai.github.io/2017/09/30/tmux/

配置tmux前缀键

在使用的时候确实会感觉C-b太不好按了，就是手要张开按，所以我也替换成了C-a

1
2
3
4
5
6

set -g prefix C-a
unbind C-b
bind C-a send-prefix

# 配置第二个前缀
set-option -g prefix2 `

修改新增面板的快捷键

1
2
3
4

unbind '"'
unbind %
bind - splitw -v -c '#{pane_current_path}'
bind | splitw -h -c '#{pane_current_path}'

# 设置窗口的开始序号
set -g base-index 1

# 设置面板的开始下标序号
set -g pane-base-index 1

# 状态栏定制
# 状态栏支持UTF8
set -g status-utf8 on

# 状态栏刷新时间
set -g status-interval 1

# 设置状态栏列表左对齐
set -g status-justify left

# 非当前窗口有内容更新在状态栏通知
setw -g monitor-activity on

# 设置状态栏前景背景色
set -g status-style "bg=black,fg=yellow"

# 设置状态栏窗口名称格式
set -wg window-status-format " #I #W"

# 设置状态栏当前窗口名称格式
set -wg window-status-current-format " #I:#W#F"

# 设置状态栏窗口名称之间的间隔
set -wg window-status-separator ""

# 设置状态栏当前窗口的样式
set -wg window-status-current-style "bg=red"

# 设置消息通知的前景背景色
set -g message-style "bg=#202529,fg=#91A8BA"

# 启用活动警告
setw -g monitor-activity on
set -g visual-activity on

# 居中窗口列表
set -g status-justify center

# 设置 PREFIX+Q 之后编号驻留的时间
set -g display-panes-time 10000

# 关闭某个窗口后，编号重排
set -g renumber-windows on

# 禁止活动进程修改窗口名
setw -g allow-rename off
# 禁止自动命名新窗口
setw -g automatic-rename off

TPM Tmux的插件管理器安裝

  1. clone整個TPM項目
1

git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
  1. 修改配置文件，使TMP生效
1
2
3
4
5

set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-sensible'

# keep this line at the very bottom of tmux.conf
run '~/.tmux/plugins/tpm/tpm'
  1. tmux重載配置
1

tmux source ~/.tmux.conf

Tmux插件

tmux-plugins/tmux-resurrect

能够保存tmux会话状态，包括面板布局等等

1
2
3

set -g @plugin 'tmux-plugins/tmux-resurrect'
set -g @resurrect-capture-pane-contents 'on'
set -g @resurrect-dir '~/.tmux/resurrect'

使用

1
2

PREFIX  C-s  # 保存tmux状态
PREFIX  C-r  # 回复tmux状态

seebi/tmux-colors-solarized

一个tmux 256的配色方案

1
2
3
4
5
6
7

set -g @plugin 'seebi/tmux-colors-solarized'

# 切换主题
set -g @colors-solarized '256'    # default
set -g @colors-solarized 'dark'
set -g @colors-solarized 'light'
set -g @colors-solarized 'base16'

tmux-plugins/tmux-prefix-highlight

这个插件在按下前缀键后可以突出显示

1

set -g @plugin 'tmux-plugins/tmux-prefix-highlight'

如果要使用需要配置状态栏

1
2
3

set -g status-right '#{prefix_highlight} #H | %a %m-%d %H:%M'
set -g @prefix_highlight_show_copy_mode 'on'
set -g @prefix_highlight_copy_mode_attr 'fg=white,bg=blue'

# 启动新的会话
tmux [new -s 会话名 -n 窗口名]

# 恢复会话
tmux at [-t 会话名]

# 列出所有的会话
tmux ls

# 关闭会话
tmux kill-session -t 会话名

# 关闭所有会话
tmux ls | grep : | cut -d. -f1 | awk '{print substr($1, 0, length($1)-1)}' | xargs kill

Tmux中的操作，需要使用前缀

会话

1
2
3
4
5
6

:new<回车> # 启动新会话
s # 列出所有会话
$ # 重命名当前会话
r # 强制重载当前会话
[ # 进入复制模式
] # 粘贴复制模式中复制的文本

窗口（标签页）

1
2

5

c   # 创建新窗口

n   # 切换后一个窗口

窗口（排序）

1
2
3

swap-window -s 3 -t 1  # 交换3和1窗口
swap-window -t 1       # 交换当前窗口和1号窗口
move-window -t 1       # 移动当前窗口到1号，依次向后排
